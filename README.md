# 🚀 Project Portfolio

This repository contains three projects developed using Java, Python, and SQL, focusing on real-world problem-solving and backend logic.

---

## 💡 SmartPay – Utility Biller (Java)

A Core Java application that calculates electricity and water bills using progressive tax slabs. It applies higher rates for increased consumption, promoting efficient resource usage.

---

## 🚖 FareCalc – Travel Optimizer (Python)

A Python-based fare calculation system for ride-sharing services. It computes ride costs based on distance, vehicle type, and surge pricing during peak hours.

---

## 🛒 Online Retail Sales Analysis Database (SQL)

A relational database project designed to store and analyze retail sales data. It uses SQL queries to generate insights such as top customers, best-selling products, and revenue trends.
